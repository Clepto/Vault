    Vault help you create and managing encrypted folders

    Project maintained by Clepto Hosted on GitHub Pages — Theme by mattgraham

    Welcome to Vault Homepage.

    Vault is a program written in Python and Gtk that make use of encfs (http://linux.die.net/man/1/encfs) and helps you create and managing encrypted folders.

    Installation:

    $ sudo apt-add-repository ppa:christriant/ppa
$ sudo apt-get update
$ sudo apt-get install vault
